# Quiz_App
Quiz App is a simple Quiz Android application with MVVM architecture and using firebase Database as the Backend .

## Used Libraries

[- Firebase](https://firebase.google.com/)

[- MVVM](https://learn.microsoft.com/en-us/xamarin/xamarin-forms/enterprise-application-patterns/mvvm)

[- Repository-Pattern](https://developer.android.com/codelabs/basic-android-kotlin-training-repository-pattern)

[- LiveData](https://developer.android.com/topic/libraries/architecture/livedata)

[- Navigation component](https://developer.android.com/guide/navigation/get-started)

[- View binding](https://developer.android.com/topic/libraries/view-binding)

[- Glide](https://github.com/bumptech/glide)
